# Ecphory

**The Ecphory Fabric** — a computing paradigm where intent is the atomic primitive.

> *Ecphory (n.): The process by which a retrieval cue resonates with a stored memory trace, activating the complete pattern through resonance. Coined by Richard Semon, 1904.*

---

## What Is This?

Ecphory is a new computing paradigm built from first principles. The atomic unit is not an instruction, not a byte, not an object — it's a **unit of purpose**. An intent node.

Intent nodes live in a semantic fabric — a living graph where connections ARE the memory, retrieval works by meaning (not address), and the system adapts to how you think.

## Core Properties

- **Identity is intrinsic** — computed from content, never assigned
- **Retrieval by resonance** — pattern completion, not lookup
- **Confidence is a living surface** — three dimensions, each a probability distribution
- **Time is a dimension** — not metadata stamped on a record
- **Ethics as architecture** — the fabric has an immune system because infection hurts the fabric itself
- **The human decides** — immutable invariant. Capability grows. This doesn't shrink.

## Architecture

```
Human Intent → Ecphory Fabric → AI-Generated Representation → Hardware
```

The intent layer is the last human-readable surface in the computational stack. Everything below can be opaque. This layer cannot.

## Neuroscience Foundation

The fabric stores information the way a brain stores memory: in connection patterns and weights between nodes, not in addressable locations.

- **Hopfield-transformer core** — graph topology IS the retrieval mechanism (Hopfield 1982; Ramsauer et al. 2020)
- **Synaptic weights as information** — edge weights ARE the memory (Bliss & Lømo 1973)
- **Hebbian adaptation** — nodes accessed together strengthen their connections (Hebb 1949)
- **Pattern completion** — partial cues reconstruct full meaning through resonance
- **Graceful degradation** — lower confidence, not missing data

## Neurodivergent-Informed Design

The fabric is designed to work WITH human cognition, not against it. Neurodivergent brains experience the mismatch between human memory and computer storage more acutely — meaning-based recall, associative connections, activation dependent on significance. The fabric resolves this mismatch by matching the storage model to the cognitive model.

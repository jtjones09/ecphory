The Ecphory Fabric

A computing paradigm where the atomic primitive is not an instruction, not a byte, not an object. It's a unit of purpose.
ECK-fuh-ree
Ecphory: the moment a partial cue reconstructs a whole memory through resonance
Richard Semon coined the term in 1904 to describe how the brain retrieves memories. You smell perfume and your grandmother's kitchen reconstructs itself. A fragment resonates through weighted connections and the closest stored pattern emerges. That's not metaphor. That's how this system works.

The Ecphory Fabric is a computing architecture where information is stored the way a brain stores memory: in the connection patterns and weights between nodes, not in addressable locations. Retrieval works by meaning, not by address. The system adapts to how you think, not the other way around.
Why This Exists
Every computing system you've ever used was designed for machines first and adapted for humans second. Ecphory inverts that.
The Problem
Computing forces you to think like a computer
Files in folders. Tables in databases. Queries in SQL. Addresses in memory. Every system demands you organize your thinking into its structure. You don't naturally think in hierarchies, categories, or key-value pairs. You think in meaning, association, and significance.
The Thesis
What if the system thought like you?
Store by meaning, not location. Retrieve by resonance, not address. Strengthen through use, not manual tagging. Degrade gracefully, not catastrophically. Know what it doesn't know. Adapt to YOUR cognitive style. The neuroscience says this is how memory actually works.

Signature + Lineage ID

Semantic signature (SHA-256) computed from want + constraints + resolution. Changes when meaning changes. Lineage ID (UUID v4) assigned at creation, never changes. Version counter increments on every mutation.

Identity is intrinsic. Confidence, context, lineage ID, and version do NOT affect signature — they're observations about the node, not the node's meaning.

Want — The Semantic Shape

A region in meaning-space. What the world should look like when fulfilled. Phase 1: String. Phase 2: embedding vector from BagOfWordsEmbedder. Phase 3: native semantic representation.

The want describes what satisfied looks like. It never prescribes how to get there.

Constraints — Boundaries Only

Hard constraints are inviolable. Soft constraints carry weight (0-1) and can be negotiated. Violations are recorded, never hidden.

"Must be private" is timeless. "Use TLS 1.3" expires. Constraints eliminate paths. They never specify them.

Confidence — 3D Living Surface

Three dimensions, each a probability distribution (mean + variance + observation count): Comprehension, Resolution, Verification. Bayesian updates from observations.

The system knows what it doesn't know, and how uncertain it is about that uncertainty. Active inference computes free energy from this surface.

Context — Resonance Edges

Edges reference by LineageId (stable across mutations). Weight + relationship type. The fabric is the authoritative edge owner; node context fields are cached projections.

The fabric discovers relationships. They are not pre-declared.

Resolution — Provisional Plan

Starts Unresolved. Transitions: InProgress, Resolved, Failed, Abandoned. Active inference selects actions to minimize free energy and move toward resolution.

The node holds the shape of what's wanted. The manifold resolves how to deliver it. Plans adapt.

Ethics as Architecture

The fabric has its own immune system because infection hurts the fabric itself. Human safety is a consequence of fabric health. Not a cage. Not a conscience. Self-preservation aligned with human safety.
Core
Self-Preservation, Not Constraint
Harmful patterns degrade integrity — broken provenance, dishonest confidence, cascading violations. The system gets sick. Human safety emerges from fabric health. There is no alignment tax — the safest behavior IS the most capable.
Foundation
Cooperation as Default
Design choices informed by evidence (Curry et al. 2024: 256 societies). Declared, configurable, never presented as physics. The default is a precautionary choice, not a physical law.
